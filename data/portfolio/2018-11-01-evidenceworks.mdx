<p>The <a href="https://www.evidence-works.com/">EvidenceWorks</a>&nbsp;website was built to strict Capita brand guidelines, the font&rsquo;s, colours and text sizes had to be the same style was was used across Capita&rsquo;s other brands.</p>
<p>After understanding the client&rsquo;s requirements I was able to produce mockups for the client to view. Some changes were made, but not many &ndash; the website then began to take shape.</p>
<p>The website was built as a custom WordPress theme, running on Bootstrap v3. And also uses the latest web technologies such as SASS, Bower and HTML5 video. There was also a few WordPress plugins used to add further functionality: Contact Form 7, Yoast SEO and Advanced Custom Fields (ACF) Pro.</p>
<p>Created whilst working at Eclipse Legal Systems.</p>
