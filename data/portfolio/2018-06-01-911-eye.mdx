<p>This website was built utilising one of Eclipse&rsquo;s <a href="/portfolio/eclipse-dynamic-webdesign-template-3/">pre-made WordPress template</a> sites, this was so the website could be pushed live in rapid time. Parts of the template were edited, such as the colour scheme, along with new additions such as the homage video header.</p>
<p>WordPress is used as the CMS and is built upon Bootstrap v2. Code was added to the &lsquo;functions.php&rsquo; to allow the use of custom navigation menus, images sizes and a custom login screen.</p>
<p>The Advanced Custom Fields plugin was also used to add extra WYSIWYG content editors for specific page templates.</p>
<p>Created whilst working at Eclipse Legal Systems.</p>
