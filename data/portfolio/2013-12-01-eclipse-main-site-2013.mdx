<p>The main objectives were for the website to stand out from its competitors and to be easy to navigate, a form is included within each sidebar for the user to contact the business.</p>
<p>Built on WordPress as a custom theme with page templates and custom post types. Other web tooling such as LESS has been used to make the CSS styles more manageable, along with Bower for managing the assortment of JavaScript libraries, and Bootstrap v2&rsquo;s responsive grid framework.</p>
