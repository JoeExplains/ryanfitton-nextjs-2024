<p>The post displays hows to use WordPress Ajax functions to display a field&#8217;s value; and automatically refresh the field every 5 seconds. This example also includes examples of how to send an Ajax post to update the field.</p>
<p>In this example, each click oft the &#8216;submit&#8217; button will increase the numerical value stored in the ACF field by 1.</p>
<p><strong>Note:</strong> All code for this example can be found in my GitHub repo: <a href="https://github.com/ryanfitton/wordpress-update-acf-field-via-ajax-example">https://github.com/ryanfitton/wordpress-update-acf-field-via-ajax-example</a>.</p>
<p>&nbsp;</p>
<h3>Prerequisites</h3>
<ul>
<li>Ensure you have WordPress installed. Tested from WordPress 5.3.2</li>
<li>Install ACF Pro (Pro required for custom options page)</li>
<li>Upload the ACF export file <code>acf-export-2020-03-31.json</code> into your ACF settings</li>
<li>Download these jQuery libaries: <a href="https://github.com/ryanfitton/wordpress-update-acf-field-via-ajax-example/blob/master/rsvp.min.js">RSVP.min.js</a> and <a href="https://github.com/ryanfitton/wordpress-update-acf-field-via-ajax-example/blob/master/js.cookie.min.js">js.cookie.min.js</a>.</li>
<li>jQuery is also required, however this should already be included from WordPress.</li>
</ul>
<p>&nbsp;</p>
<h3>Explanation of files</h3>
<ul>
<li><code>tmpl_page_ajax.php</code> &#8211; Page template</li>
<li><code>functions.php</code> &#8211; Functions for Ajax calls</li>
<li><code>ajax.js</code> &#8211; JS Ajax calls</li>
</ul>
<p>&nbsp;</p>

    /***********************
    	Ajax Counter Increase - Update counter
    ***********************/
    function ajax_counter_increase() {

    	/*
    		Nonce Security check
    	*/
    	if ( ! wp_verify_nonce( $_POST['nonce'], 'nonce' ) ) {
    		wp_send_json_error();
    	}


    	//Get current field value
    	$get_field_value = get_field('counter_val', 'options_counter');


    	//If current field is found
    	if ( $get_field_value != '' ) {

    		//Increase the value by 1
    		$increase_value = 1;


    		//New value
    		$new_value = $get_field_value + $increase_value;

    		//Update post meta value with new value
    		if ( update_field( 'counter_val', $new_value, 'options_counter') ) {

    			//Data to return via JSON
    			$success = array(
    				'value' =&gt;  esc_html( $new_value ),
    			);

    		} else {
    			$error = array( 'value' =&gt; "Value could not be updated" );
    		}

    	} else {
    		$error = array( 'value' =&gt; "Cannot retrieve value" );
    	}


    	/*
    		Return Data
    	*/
    	//If Error
    	if ( !empty($error) ) {
    		wp_send_json_error($error);

    	//If success
    	} elseif ( !empty($success) ) {
    		wp_send_json_success( $success );

    	//Catch all
    	} else {
    		wp_send_json_error();
    	}


    	die();
    }
    //Action: 'counter_increase' posted in AJAX, runs this function: 'ajax_counter_increase'
    add_action('wp_ajax_counter_increase', 'ajax_counter_increase');		//Run for Admin users
    add_action('wp_ajax_nopriv_counter_increase', 'ajax_counter_increase');	//Run for Non-Admin users
