This is a quick, but I thought it would be worth a post.

Within Linux (I'm using Ubuntu 17.04) you can quickly export your
computer specs as an HTML file, useful if you need some info on what
particular hardware you're running.

Simply run this within your terminal at the location where you want the
.html file to be created:

``` language-markup
sudo lshw -html > specs.html
```
