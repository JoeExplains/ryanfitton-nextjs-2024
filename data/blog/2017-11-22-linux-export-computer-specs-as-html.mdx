<p>This is a quick, but I thought it would be worth a post.</p>
<p>
  Within Linux (I&#8217;m using Ubuntu 17.04) you can quickly export your computer specs as an HTML
  file, useful if you need some info on what particular hardware you&#8217;re running.
</p>
<p>
  Simply run this within your terminal at the location where you want the .html file to be created:
</p>
```
sudo lshw -html > specs.html
```
