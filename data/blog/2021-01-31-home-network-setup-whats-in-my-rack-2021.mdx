<p>Following on from my <a href="https://ryanfitton.co.uk/blog/how-i-installed-cat6-ethernet-in-my-home/">original post</a> in 2017 and a <a href="https://ryanfitton.co.uk/blog/home-network-setup-whats-in-my-rack-2019/">follow up</a> I made in 2019, I am making this post to show the changes my to my home network.</p>
<p>Some parts have not changed. such as;</p>
<ul>
<li>The rack, a 9U with dimensions of 600x600mm.</li>
<li>I am still running pfSense on a <a href="https://www.aliexpress.com/item/32863982193.html?spm=a2g0s.9042311.0.0.27424c4dp3dCSF">Qotom Q150P-S08</a> Mini PC.</li>
<li>Still using a HP ProCurve 24 Port Gigabit Switch (<a href="https://h10057.www1.hp.com/ecomcat/hpcatalog/specs/provisioner/05/J9028B.htm">1800-24G J9028B</a>) as my core network switch.</li>
<li>And <a href="https://www.amazon.co.uk/APC-Back-UPS-BX-Uninterruptible-Shutdown/dp/B00T7BYV6W/ref=as_li_ss_tl?keywords=apc+700&amp;qid=1559998754&amp;s=computers&amp;sr=1-4&amp;linkCode=ll1&amp;tag=ryanfittoncou-21&amp;linkId=4b82a7af15071bab059cfbeeb8f3d427&amp;language=en_GB">APC Back-UPS</a> (BX700UI) UPS.</li>
</ul>
<p>Although I have made changes to the LXD containers running on my home server. I am still running <a href="https://gitlab.com/Shinobi-Systems/Shinobi">Shinobi DVR</a>, Cups as a print server and Samba shares for network storage access. <a href="https://www.freepbx.org/">FreePBX</a> has been removed and <a href="https://emby.media/">Emby</a> has been replaced with <a href="https://www.plex.tv/en-gb/">Plex</a>. These are new containers: <a href="https://landscape.canonical.com/set-up-on-prem">Ubuntu Landscape</a>, <a href="https://www.home-assistant.io/">Home Assistant</a>, <a href="https://www.librenms.org/">LibreNMS</a> and <a href="https://ryanfitton.co.uk/blog/installing-tp-link-omada-controller-on-ubuntu/">TP-Link Omada</a> controller software.</p>
<p>I have also purchased new hardware;</p>
<ul>
<li>New PoE+ switch: HP 1920-24G-PoE+ (180W) JG925A &#8211; This will be replacing a TP-Link PoE switch.</li>
<li>In the garage I have installed a new PoE+ switch: Netgear GS110TP and a TP-Link EAP 225 Outdoor access point <a href="https://www.amazon.co.uk/TP-Link-MU-MIMO-Wireless-Gigabit-Outdoor/dp/B07C217N5V?&amp;linkCode=ll1&amp;tag=ryanfittoncou-21&amp;linkId=2c37be153584a9d114851b91f523b414&amp;language=en_GB&amp;ref_=as_li_ss_tl" target="_blank" rel="noopener">TP-Link EAP 225 Outdoor access point</a> (currently indoors, but I hope to mount this outside soon)</li>
<li>Switched broadband providers to BT. I am using an basic Openreach branded Huawei VDSL HG612 modem, the pfSense router handles PPPoE connectivity.</li>
<li>I&#8217;ve also purchased a <a href="https://www.amazon.co.uk/gp/product/B07ZFSX8LB?ie=UTF8&amp;psc=1&amp;linkCode=ll1&amp;tag=ryanfittoncou-21&amp;linkId=4e46eb6fe19caf5e242adbfe0685cc59&amp;language=en_GB&amp;ref_=as_li_ss_tl">Hauwei B535-232 4G router</a> to act as a backup internet connection, pfSense automatically handles failover.</li>
<li>A temperature monitor using a <a href="https://www.amazon.co.uk/Raspberry-Pi-Zero-WH/dp/B07BHMRTTY/ref=as_li_ss_tl?dchild=1&amp;keywords=raspberry+pi+zero&amp;qid=1609874665&amp;s=computers&amp;sr=1-6&amp;linkCode=ll1&amp;tag=ryanfittoncou-21&amp;linkId=a7fe57b67a193551368f70f074281266&amp;language=en_GB">Raspberry Pi Zero</a> with sensor <a href="https://ryanfitton.co.uk/blog/building-a-raspberry-pi-zero-temperature-monitor-with-web-interface/">running a custom web UI</a> I have developed.</li>
<li>Also in the garage I have built a NAS powered by a 4GB Raspberry Pi 4, images below.</li>
</ul>

<p>Network VLANs have been setup to help improve network security. This is one of the reasons why I purchased two new network switches. Details of these VLANs are below:</p>
<ul>
<li>Lan (Default Lan network &#8211; no VLAN, had access to all VLANs)</li>
<li>VLAN10 &#8211; Secure network for servers (can access other VLANs, can only be accessed from LAN)</li>
<li>VLAN20 &#8211; IP Cam network (No access to other VLANs, Internet access blocked)</li>
<li>VLAN30 &#8211; IoT network (No access to other VLANs)</li>
<li>VLAN40 &#8211; Guest network (No access to other VLANs, Internet access has been speed limited)</li>
</ul>
