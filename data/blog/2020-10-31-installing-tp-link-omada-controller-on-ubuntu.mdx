<h3>Getting started</h3>
<p><a href="/static/img/blog/2020/10/omada_screenshot_2.png"><img loading="lazy"  src="/static/img/blog/2020/10/omada_screenshot_2-300x144.png" alt="" width="300" height="144" srcset="/static/img/blog/2020/10/omada_screenshot_2-300x144.png 300w, /static/img/blog/2020/10/omada_screenshot_2-1024x491.png 1024w, /static/img/blog/2020/10/omada_screenshot_2-768x368.png 768w, /static/img/blog/2020/10/omada_screenshot_2.png 1308w" sizes="(max-width: 300px) 100vw, 300px" /></a>This tutorial will install the Omada controller on Ubuntu. In my case, this was Ubuntu 18.04, and the Omada Controller version v3.2.10.</p>
<p>I&#8217;m running this in an Ubuntu LXD container, but I imagine this would work fine on a bare-metal install.</p>
<h3>Installation</h3>
<ol>
<li style="list-style-type: none;">
<ol>
<li>Update your packages:
`sudo apt-get update`
</li>
<li>Install Mongo DB:
`sudo apt-get install mongodb`
</li>
<li>Install JSVC, a tool to launch Java applications as daemons:
`sudo apt-get install jsvc`
</li>
<li>Install Open JDK v8:
`sudo apt-get install openjdk-8-jdk`
</li>
<li>Configure system to use ‘&#8230; 8-openjdk &#8230;’, usually this is option 2:
`sudo update-alternatives --config java`
</li>
<li>
<p>Now, we have installed the prerequisites, you now need to download the Omada controller package.In this tutorial the ‘.deb’ packages will be installed, these offer the most easiest way to install the Omada controller software.</p>
<p>Download the latest Omada controller software from TP-Link&#8217;s website here: <a href="https://www.tp-link.com/uk/support/download/omada-software-controller/#Controller_Software" target="_blank" rel="noopener noreferrer">https://www.tp-link.com/uk/support/download/omada-software-controller/#Controller_Software</a>.</p>
<p>In this example, I am using version 3.2.10.Download the package via WGET:</p>
`sudo wget https://static.tp-link.com/2020/202004/20200420/omada_v3.2.10_linux_x64_20200330135119.deb`
</p>
</li>
<li>Install the package:
`sudo dpkg –i omada_v3.2.10_linux_x64_20200330135119.deb`
</li>
<li>The installation will then guide you through the rest of the installation. Once complete, you will be prompted to go to a web address to finish setup. E.g. http://LOCALHOST/YOUR IP ADDRESS:8088.</li>
<li>You can start/stop the controller software with these commands:Start the controller. By default your controller should run at start-up, but try this for a manual start:
`sudo tpeap start`
<p>Stop the controller:</p>
`sudo tpeap stop`
<p>See the status the running controller:</p>
`sudo tpeap status`
</li>
</ol>
</li>
</ol>
