This is a follow on from my first post which described how I installed multiple CAT6 network cable runs in my house, view [here](/blog/blog/how-i-installed-cat6-ethernet-in-my-home/).

In this post, I will talk through the equipment which is currently in my network rack for 2019.

Firstly, I have upgraded to a larger rack; now a 9U 600x600mm rack.

Here is a list of the hardware from top to bottom:

1.  Top row (Left): [Qotom Q150P-S08](https://www.aliexpress.com/item/32863982193.html?spm=a2g0s.9042311.0.0.27424c4dp3dCSF) Mini PC running [pfSense](https://www.pfsense.org/) -- Runs my network's routing functions and VPN, and has support AES-NI encryption which will in come useful for later versions of pfSense. The processor is a Quad Core Celeron N3160 with 2x 1GB Ethernet ports, small 16GB SSD and 2GB DDR3 memory.
2.  Top row (Right): Virgin Media -- Running in modem mode. Provided by Virgin Media, an ISP in the UK.
3.  HP ProCurve 24 Port Gigabit Switch ([1800-24G J9028B](https://h10057.www1.hp.com/ecomcat/hpcatalog/specs/provisioner/05/J9028B.htm)) -- The main switch, all devices connect into this.
4.  TP-Link 8 port Gigabit POE switch ([TL-SG1008PE](https://www.amazon.co.uk/dp/B00CGY449G/ref=as_li_ss_tl?ie=UTF8&linkCode=ll1&tag=ryanfittoncou-21&linkId=38ca8c0cb2bd050b6cc6e0f02e28d441&language=en_GB)) -- POE switch, mainly for WiFi access points and IP Cams.
5.  Custom built server in a short depth [3U Logic Case chassis](https://www.logic-case.com/products/rackmount-chassis/3u/3u-short-depth-chassis-w-3-x-525-drive-bays-sc-34390/), the case was not designed to be rack mounted but I was able to mount is with these rails [here](https://www.scan.co.uk/products/logic-sc-03a-server-case-500mm-rail-kit-for-2u-3u-and-4u-max-load-35kg), just needed some small bolts and extra holes to be drilled. The server is running an Intel i3-4170 with a Crucial (CT2KIT102472BD160B) 16GB DDR3 ECC memory kit, Supermicro X10SLL-F motherboard, 2x 3TB WD RED drives (mirrored raid) and Intel (EXPI9404PT) Pro/1000 PT Quad Port Ethernet Adaptor. I am running Ubuntu Server with LXD containers for my software: [Shinobi DVR](https://gitlab.com/Shinobi-Systems/Shinobi), Cups print server, [FreePBX](https://www.freepbx.org/), [Emby](https://emby.media/) and Samba storage.
6.  Also; not in the image, there is an [APC Back-UPS](https://www.amazon.co.uk/APC-Back-UPS-BX-Uninterruptible-Shutdown/dp/B00T7BYV6W/ref=as_li_ss_tl?keywords=apc+700&qid=1559998754&s=computers&sr=1-4&linkCode=ll1&tag=ryanfittoncou-21&linkId=4b82a7af15071bab059cfbeeb8f3d427&language=en_GB) (BX700UI) which can last around 10 minutes powering everything but the laser printer.
