<p>
  This is a follow on from my first post which described how I installed multiple CAT6 network cable
  runs in my house, view{' '}
  <a href="https://ryanfitton.co.uk/blog/how-i-installed-cat6-ethernet-in-my-home/">here</a>.
</p>
<p>
  In this post, I will talk through the equipment which is currently in my network rack for 2019.
</p>
<p>Firstly, I have upgraded to a larger rack; now a 9U 600x600mm rack.</p>

<p>Here is a list of the hardware from top to bottom:</p>
<ol>
  <li>
    Top row (Left):{' '}
    <a href="https://www.aliexpress.com/item/32863982193.html?spm=a2g0s.9042311.0.0.27424c4dp3dCSF">
      Qotom Q150P-S08
    </a>{' '}
    Mini PC running <a href="https://www.pfsense.org/">pfSense</a> &#8211; Runs my network&#8217;s
    routing functions and VPN, and has support AES-NI encryption which will in come useful for later
    versions of pfSense. The processor is a Quad Core Celeron N3160 with 2x 1GB Ethernet ports,
    small 16GB SSD and 2GB DDR3 memory.
  </li>
  <li>
    Top row (Right): Virgin Media &#8211; Running in modem mode. Provided by Virgin Media, an ISP in
    the UK.
  </li>
  <li>
    HP ProCurve 24 Port Gigabit Switch (
    <a href="https://h10057.www1.hp.com/ecomcat/hpcatalog/specs/provisioner/05/J9028B.htm">
      1800-24G J9028B
    </a>
    ) &#8211; The main switch, all devices connect into this.
  </li>
  <li>
    TP-Link 8 port Gigabit POE switch (
    <a href="https://www.amazon.co.uk/dp/B00CGY449G/ref=as_li_ss_tl?ie=UTF8&amp;linkCode=ll1&amp;tag=ryanfittoncou-21&amp;linkId=38ca8c0cb2bd050b6cc6e0f02e28d441&amp;language=en_GB">
      TL-SG1008PE
    </a>
    ) &#8211; POE switch, mainly for WiFi access points and IP Cams.
  </li>
  <li>
    Custom built server in a short depth{' '}
    <a href="https://www.logic-case.com/products/rackmount-chassis/3u/3u-short-depth-chassis-w-3-x-525-drive-bays-sc-34390/">
      3U Logic Case chassis
    </a>
    , the case was not designed to be rack mounted but I was able to mount is with these rails{' '}
    <a href="https://www.scan.co.uk/products/logic-sc-03a-server-case-500mm-rail-kit-for-2u-3u-and-4u-max-load-35kg">
      here
    </a>
    , just needed some small bolts and extra holes to be drilled. The server is running an Intel
    i3-4170 with a Crucial (CT2KIT102472BD160B) 16GB DDR3 ECC memory kit, Supermicro X10SLL-F
    motherboard, 2x 3TB WD RED drives (mirrored raid) and Intel (EXPI9404PT) Pro/1000 PT Quad Port
    Ethernet Adaptor. I am running Ubuntu Server with LXD containers for my software:{' '}
    <a href="https://gitlab.com/Shinobi-Systems/Shinobi">Shinobi DVR</a>, Cups print server,{' '}
    <a href="https://www.freepbx.org/">FreePBX</a>, <a href="https://emby.media/">Emby</a> and Samba
    storage.
  </li>
  <li>
    Also; not in the image, there is an{' '}
    <a href="https://www.amazon.co.uk/APC-Back-UPS-BX-Uninterruptible-Shutdown/dp/B00T7BYV6W/ref=as_li_ss_tl?keywords=apc+700&amp;qid=1559998754&amp;s=computers&amp;sr=1-4&amp;linkCode=ll1&amp;tag=ryanfittoncou-21&amp;linkId=4b82a7af15071bab059cfbeeb8f3d427&amp;language=en_GB">
      APC Back-UPS
    </a>{' '}
    (BX700UI) which can last around 10 minutes powering everything but the laser printer.
  </li>
</ol>
