<p>I thought I would share this simple PHP function to hopefully help anyone needing a quick solution.</p>
<p>Imagine, you have a multi-dimensional array such as this:</p>
<pre class="language-php"><code class="language-php">&lt;?php
    //Example array
    $array = array(
      'test' =&gt; array(
          'name' =&gt; 'John Doe',
          'email' =&gt; 'john@example.com'
      )
    );
?&gt;</code></pre>
<p>And you need to extract a specific bit of data, such as a name or an email address. You already know what the key name will be but require the value stored against it.</p>
<p>Run this function, pass the array and the name of key you&#8217;re searching for:</p>
<pre class="language-php"><code class="language-php">&lt;?php
    /*
        Function: Find a key's value pair

        $array = The array data to look through
        $findKey = The Key name to be searched for. The value pair of this key will be returned.
    */
    function keyValue($array, $findKey) {

        //Loop through array
        foreach ($array as $key =&gt; $value) {
            
            //If the key of this current loop matched what we're looking for, return the value
            if ($key == $findKey) {
                $output = $value;

            //If still an array, it must be multi-dimensional, re-look through array
            } elseif (is_array($value)) {
                $output = keyValue($value, $findKey);
            }
        }

        return $output;
    }
?&gt;</code></pre>
<p>The result will return the value that was stored against the key. Here is a full code example:</p>
