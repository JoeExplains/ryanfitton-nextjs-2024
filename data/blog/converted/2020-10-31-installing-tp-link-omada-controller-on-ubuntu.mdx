work fine on a bare-metal install.

### Installation

1.  1.  Update your packages:

        ``` {.language-markup .code-toolbar}
        sudo apt-get update
        ```

    2.  Install Mongo DB:

        ``` {.language-markup .code-toolbar}
        sudo apt-get install mongodb
        ```

    3.  Install JSVC, a tool to launch Java applications as daemons:

        ``` {.language-markup .code-toolbar}
        sudo apt-get install jsvc
        ```

    4.  Install Open JDK v8:

        ``` {.language-markup .code-toolbar}
        sudo apt-get install openjdk-8-jdk
        ```

    5.  Configure system to use '... 8-openjdk ...', usually this is
        option 2:

        ``` {.language-markup .code-toolbar}
        sudo update-alternatives --config java
        ```

    6.  Now, we have installed the prerequisites, you now need to
        download the Omada controller package.In this tutorial the
        '.deb' packages will be installed, these offer the most easiest
        way to install the Omada controller software.

        Download the latest Omada controller software from TP-Link's
        website here:
        [https://www.tp-link.com/uk/support/download/omada-software-controller/#Controller_Software](https://www.tp-link.com/uk/support/download/omada-software-controller/#Controller_Software){target="_blank"
        rel="noopener noreferrer"}.

        In this example, I am using version 3.2.10.Download the package
        via WGET:

        ``` {.language-markup .code-toolbar}
        sudo wget https://static.tp-link.com/2020/202004/20200420/omada_v3.2.10_linux_x64_20200330135119.deb
        ```

    7.  Install the package:

        ``` {.language-markup .code-toolbar}
        sudo dpkg –i omada_v3.2.10_linux_x64_20200330135119.deb
        ```

    8.  The installation will then guide you through the rest of the
        installation. Once complete, you will be prompted to go to a web
        address to finish setup. E.g. http://LOCALHOST/YOUR IP
        ADDRESS:8088.

    9.  You can start/stop the controller software with these
        commands:Start the controller. By default your controller should
        run at start-up, but try this for a manual start:

        ``` {.language-markup .code-toolbar}
        sudo tpeap start
        ```

        Stop the controller:

        ``` {.language-markup .code-toolbar}
        sudo tpeap stop
        ```

        See the status the running controller:

        ``` {.language-markup .code-toolbar}
        sudo tpeap status
        ```

### Final Notes

This tutorial can be installed on any Ubuntu running computer; Container
(as I do) or Bare-Metal device. If you don't have the ability to run
this software on your own hardware, TP-Link have their own device to run
the controller software; the [Omada Cloud Controller
OC200](https://www.tp-link.com/us/business-networking/omada-sdn-controller/oc200/){target="_blank"
