<p>I&#8217;ve decided to share this code for anyone who requires a quick &#8216;copy and paste&#8217; solution for using data maps (map-get) and loops within Sass.</p>
<p>This example will allow you to set values within the data map, then loop through and output as CSS rules.</p>
<p>Setup the variables within the data map and create the code for the loop:</p>
<pre class="language-scss"><code class="language-scss">//Badges
$badges: (
    //Badge 1
    1: (
        background: #54677b,
        color: #f1ec69
    ),
    
    //Badge 2
    2: (
        background: #ecebeb,
        color: #ffffff
    ),
    
    //Badge 3
    3: (
        background: #30b59f,
        color: #2f343e
    )
);

//Loop
@each $name, $colors in $badges {
    #badge-#{$name} {
background:map-get($colors,background);
        color:map-get($colors,color);
}
}</code></pre>

<p>The compiled CSS will be returned as:</p>
<pre class="language-css"><code class="language-css">#badge-1 {
  background: #54677b;
  color: #f1ec69;
}

#badge-2 {
background: #ecebeb;
color: #ffffff;
}

#badge-3 {
background: #30b59f;
color: #2f343e;
}

</code></pre>
