<p>Today I found out you can easily notify Google of an XML sitemap by &#8216;pinging&#8217; Google via a GET request:</p>
<pre class="language-markup"><code class="language-markup">http://www.google.com/webmasters/sitemaps/ping?sitemap=URLOFSITEMAP.xml</code></pre>
<p>You can perform this via PHP, or if running on Linux; use wget.</p>
