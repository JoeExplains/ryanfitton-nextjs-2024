sizes="(max-width: 194px) 100vw, 194px"}](/static/img/blog/2020/07/41lZcMOKJXL._SX425_.jpg)I've
recently purchased a few cheap DHT22 temperature sensors from
[Amazon](https://www.amazon.co.uk/gp/product/B07L83FFV8/ref=as_li_ss_tl?ie=UTF8&psc=1&linkCode=ll1&tag=ryanfittoncou-21&linkId=e0231675027ea1f10a361eb064611f46&language=en_GB).

They work great and use Python to display the data from the sensors.

I found I required a way to log the data, so I though why not make this
a project!?

I have created a simple Web Interface to view logged data:
<https://github.com/ryanfitton/rpi-temperature-monitor-logging-interface>.

This project is built using PHP, Python3 and Bootstrap for nice admin
styling.

The project's [GitHub
page](https://github.com/ryanfitton/rpi-temperature-monitor-logging-interface)
gives much more information on how to get-up-and-running.
