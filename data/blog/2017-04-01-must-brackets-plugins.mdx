<p>As a follow on from my previous post &#8216;<a href="https://ryanfitton.co.uk/blog/monokai-theme-for-brackets/">Monokai theme for Brackets</a>&#8216;, I have produced a list of my must-have plugins for <a href="http://brackets.io/">Adobe Brackets</a>. Let&#8217;s go!</p>
<ul>
<li><strong>Brackets Selections</strong><br />
This plugin will add Sublime Text style selections.<br />
Github: <a href="https://github.com/DannyMoerkerke/brackets-selections">https://github.com/DannyMoerkerke/brackets-selections</a></li>
<li><strong>Brackets Sort Text Extension</strong><br />
Provides a couple of utility functions for sorting lines in the current document.<br />
Github: <a href="https://github.com/ingorichter/brackets-sort-text/">https://github.com/ingorichter/brackets-sort-text/</a></li>
<li><strong>Tabs &#8211; Custom Working</strong><br />
This plugin will add tabs for currently open files.<br />
Github: <a href="https://github.com/DH3ALEJANDRO/custom-work-for-brackets">https://github.com/DH3ALEJANDRO/custom-work-for-brackets</a></li>
<li><strong>Indent Guides for Brackets</strong><br />
Show indent guides in the code editor.<br />
Github: <a href="https://github.com/lkcampbell/brackets-indent-guides">https://github.com/lkcampbell/brackets-indent-guides</a></li>
<li><strong>Brackets Minimap</strong><br />
Adds a scrollable minimap, similar to Sublime Text<br />
Github: <a href="https://github.com/zorgzerg/brackets-minimap">https://github.com/zorgzerg/brackets-minimap</a></li>
<li><strong>Quick multicursor selection</strong><br />
Adds the extremely useful multiple selection tool. This is done in Sublime Text with <kbd><kbd>CTRL</kbd> + <kbd>D</kbd></kbd>, however in this Brackets version use <kbd><kbd>ALT</kbd> + <kbd>D</kbd></kbd>.<br />
Github: <a href="https://github.com/louiealmeda/multiple-cursor-select">https://github.com/louiealmeda/multiple-cursor-select</a></li>
</ul>
