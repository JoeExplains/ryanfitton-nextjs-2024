The [WordPress 3.7 \[Basie\]](http://wordpress.org/news/2013/10/basie/) release went live on the 24 October 2013. It's great release and includes a number of important features, one of which is Automatic Updates.

By default automatic updates will only be applied to minor version upgrades and not major releases. For example; versions such as 3.7.1 to 3.7.2 will be applied automatically, however upgrades to major versions such as 3.7.2 to 3.8 will not be applied. \[ [Source](http://www.wpbeginner.com/wp-tutorials/how-to-disable-automatic-updates-in-wordpress/) \].

As mentioned above, there are different types of WordPress updates:

-   WordPress Core code updates
-   Minor WordPress core updates, maintenance and security
-   Major WordPress core code

You can change how WordPress applies updates by using a specific line of code. Different options for updates are shown below, these should be placed in your wp-config.php file.
