<p>The <a href="http://wordpress.org/news/2013/10/basie/">WordPress 3.7 [Basie]</a> release went live on the 24 October 2013. It&#8217;s great release and includes a number of important features, one of which is Automatic Updates.</p>
<p>By default automatic updates will only be applied to minor version upgrades and not major releases. For example; versions such as 3.7.1 to 3.7.2 will be applied automatically, however upgrades to major versions such as 3.7.2 to 3.8 will not be applied. [<a href="http://www.wpbeginner.com/wp-tutorials/how-to-disable-automatic-updates-in-wordpress/">Source</a>].</p>
<p>As mentioned above, there are different types of WordPress updates:</p>
<ul>
<li>WordPress Core code updates</li>
<li>Minor WordPress core updates, maintenance and security</li>
<li>Major WordPress core code</li>
</ul>
